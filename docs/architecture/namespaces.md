# Namespace Strategy

platform
Purpose:
Infrastructure services used by the platform itself.

Examples:
- Metrics Server
- Platform utilities

--------------------------------------------------

app
Purpose:
Business application workloads.

Contains:
- React frontend
- Go API
- MongoDB

--------------------------------------------------

monitoring
Purpose:
Observability stack.

Contains:
- Prometheus
- Grafana

--------------------------------------------------

ingress
Purpose:
External traffic management.

Contains:
- Ingress Controller
