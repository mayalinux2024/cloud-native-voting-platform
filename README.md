# Cloud-Native Voting Platform

## Project Overview

This project demonstrates how to build and operate a production-style cloud-native application on Amazon EKS.

Rather than focusing only on deploying containers, the project showcases Infrastructure as Code, Kubernetes operations, CI/CD automation, monitoring, and production best practices.
